Playing Codenames
Code Masters
  1. Go to the Codemasters page. Regular and Adult Codenames words can be toggled using the Clean It Up / Get Dirty button.
  2. One codemaster begins a game by clicking Start New Game and shares the board number with the other.
  3. The other codemaster joins by typing in the board number and clicking View Existing Boards.
  4. Guessed words can be coverd using the drop down menus. Reminder that rows and horizontal and columns are vertical. The top row is row 1 and the leftmost column is column 1.
  5. Use the undo button in case of mistakes in covering cards.
Everyone Else
  1. Go to the Guessers page to see the Codenames words.
  2. When a codemaster covers a word to indicate its true team, refresh your browser to see the changes.
